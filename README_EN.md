# ASS TO LRC
[[简体中文](https://github.com/xiaojiang1727/ass-to-lrc/blob/main/README.md)|[English](https://github.com/xiaojiang1727/ass-to-lrc/blob/main/README_EN.md)]
>A lightweight, zero-dependency Python script specifically designed to convert .ass subtitle files with \k, \K, \kf tags into enhanced .lrc lyric files.
- **Note: This script is generated by AI, manually modified, and includes detailed comments.**

## Core Features
- Word-Level Timestamps: Perfectly parses \k, \K, \kf tags in ASS to generate enhanced LRC format with word-by-word highlighting
- Intelligent Text Cleaning: Automatically removes ASS tags and converts \N, \n line breaks into smooth spaces
- High Compatibility: Automatically handles UTF-8 and UTF-8 with BOM encoding to avoid garbled text.
- Zero Dependencies: Uses only Python standard libraries, no third-party packages required.
